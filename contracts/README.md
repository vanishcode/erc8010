# ERC-8010 Smart Contracts

This directory contains the Solidity smart contracts and Foundry tests for the ERC-8010 demo.

For detailed documentation on the contract design, storage layouts, and testing strategy, please refer to:
**[docs/architecture/contracts.md](../docs/architecture/contracts.md)**

## Quick Start

```bash
# Compile
forge build

# Test
forge test -vvv
```
